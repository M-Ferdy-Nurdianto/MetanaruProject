## 🎸 Features

- ✅ **Dynamic Member Profiles**: Fay, Joy, Caca, Nera (dengan efek Gold Glow).
- ✅ **Advanced Cheki System**: Booking Solo & Group dengan kalkulasi otomatis.
- ✅ **Secure Backend**: Integrasi Supabase Auth & Database.
- ✅ **Admin Dashboard**: Manajemen order, event, dan pengaturan harga secara real-time.
- ✅ **Premium Aesthetics**: Tema Dark Metal, Glassmorphism, dan Micro-animations.
- ✅ **SEO Optimized**: Meta tags dinamis untuk setiap halaman.

## 🚀 Deployment

Project ini dioptimalkan untuk dideploy di **Vercel**.

### Environment Variables Required:
- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
